ModuBot

A powerful, modular Discord bot built for server management and engagement

Add to Discord Learn More

Features

Moderation

Auto-delete spam, temporary mutes, strikes system, and detailed moderation logs to keep your server safe.

Entertainment

Mini-games like trivia and hangman, meme generator, dice rolling, and 8-ball predictions to keep your members engaged.

Utility

Server information, polls, reminders, weather updates, and customizable commands for ultimate flexibility.

Customization

Custom prefix, server-specific settings, and role-based permissions to tailor the bot to your server's needs.

Economy System

Complete virtual economy with currency, daily rewards, gambling games, leaderboards, and a shop system. Learn more.

Slash Commands

Modern Discord slash commands with autocomplete and detailed help, alongside traditional prefix commands.

Popular Commands

/mod ban @user [reason] [delete_days]
Ban a user from the server with optional reason and message deletion
/mod kick @user [reason]
Kick a user from the server with optional reason
/mod timeout @user <duration> [reason]
Temporarily timeout a user for specified duration
/mod clear <amount> [user] [contains]
Delete a specified number of messages with optional filters
!strike @user [reason]
Give a warning strike to a user
!trivia [category]
Start a trivia game with optional category
!hangman
Start a game of hangman
!meme
Get a random meme from Reddit
!8ball [question]
Ask the magic 8-ball a question
!roll [dice]
Roll dice (e.g., 2d6 for two six-sided dice)
/serverinfo
Display detailed information about the server
/userinfo [@user]
Show information about a user or yourself
/utility weather [location]
Get current weather information for a location
!poll [duration] [question] [options...]
Create a poll with multiple options
!remind [time] [reminder]
Set a reminder for a specific time
/economy balance [user]
Check your or another user's balance
/economy daily
Claim your daily currency reward
/economy shop
View items available in the shop
/economy buy <item_id>
Purchase an item from the shop
/gamble coinflip <amount> <choice>
Bet on a coin flip to win double your bet
!cc add [name] [response]
Create a custom command
!cc remove [name]
Remove a custom command
!cc edit [name] [new response]
Edit an existing custom command
!cc list
List all custom commands for the server
!cc info [name]
Get details about a specific custom command

Frequently Asked Questions

Is ModuBot free to use?

Yes, ModuBot is completely free to use with all features available to everyone. There are no premium tiers or paywalls.

Can I customize the bot's prefix?

Yes, you can set a custom prefix for your server using the !setprefix command. For example, !setprefix ? would change the prefix to ?.

How do I set up auto-moderation?

Use the moderation commands to set up auto-moderation. You can configure banned words, anti-spam protection, and auto-muting with various commands. Check !help moderation for details.

How does the economy system work?

ModuBot has a comprehensive economy system where users can earn daily rewards, gamble with games like coinflip and slots, purchase items from the shop, and more. Check out the Economy Guide for all the details.

Can I host this bot myself?

Yes, ModuBot is open-source and can be self-hosted. You can find the source code on GitHub along with setup instructions.

What permissions does the bot need?

For full functionality, ModuBot needs permissions to read and send messages, manage messages and channels, kick and ban members, and manage roles. Some features will work with fewer permissions.
