Command Overview

ModuBot uses Discord's slash commands for a clean, intuitive interface. Here are some of the most common commands:

  • /mod ban [user] [reason] [delete_days]

    Ban a user from the server with optional message deletion period

  • /mod kick [user] [reason]

    Kick a user from the server

  • /mod timeout [user] [duration] [reason]

    Timeout a user for a specified duration

  • /global profile [user]

    View your or another user's global profile

  • /global shop

    View the global shop for passive income items

  • /global collect

    Collect income from all your passive income sources
